Introduction

Blast-off is a simple project template that includes a basic HTML structure, a CSS reset, a grid system, and some build tools to help you get started with your project. A lot of effort is put into keeping code simple, it's encouraged to dive into the files and inspect the code in order to learn how this all works.

Grid system

Blast-off comes packaged with a simple grid system, inspect the HTML to learn about how this is set up.

JavaScript

Included in this project are some handy JavaScript files, including event dispatching and localStorage helpers.

Commands

Blast-off uses parcel.js to package everything into a lovely little website. Please refer to the package.json file to check out the available commands.
